-------------

dice::

  usage: lung [-h] [-s] [-f F [F ...]] [-l L [L ...]] [-m M [M ...]] [-c [C]]
              [-qs [QS]] [-g G [G ...]] [-lw]

  optional arguments:
    -h, --help    show this help message and exit
    -s            sequential (non random)
    -f F [F ...]  files
    -l L [L ...]  lists
    -m M [M ...]  modules
    -c [C]        correct retries
    -qs [QS]      cuestion count
    -g G [G ...]  question grep
    -lw           lock weight
